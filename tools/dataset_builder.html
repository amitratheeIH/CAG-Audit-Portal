Paste your table
Copy the entire table from Word or PDF (select all cells → Ctrl+C) and paste it below.
The first row is treated as column headers. Columns are separated by tabs (Word) or multiple spaces (PDF).
Handles tab-separated (Word), comma-separated (CSV), or space-aligned (PDF). Nil markers (—, -, N.A., nil) auto-detected.
From Word: Click inside the table → Ctrl+A → Ctrl+C → paste here.
From PDF: Use Acrobat's "Export to Excel" or copy-paste — space-aligned columns are handled automatically.
From Excel/CSV: Select the cells → Ctrl+C — tab-separated paste works perfectly.
